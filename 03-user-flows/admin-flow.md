# OneFarmTech Admin Flow

## Admin Responsibilities

The admin dashboard is the operational centre of OneFarmTech.

Admin should be able to:

- Create and manage customers
- Create and manage orders
- Add products and produce categories
- Record farmer/supplier details
- Track payment status
- Track sourcing and fulfilment status
- Create group-buy offers
- Manage pickup locations
- Manage delivery options
- Record complaints and resolutions
- Approve or reject credit/payment terms for business buyers

## Manual Order Creation Flow

1. Admin receives WhatsApp order.
2. Admin checks buyer details.
3. Admin creates customer or selects existing customer.
4. Admin creates order.
5. Admin adds produce items, quantities, grades, and prices.
6. Admin selects order type.
7. Admin selects delivery or pickup method.
8. Admin sets payment requirement.
9. Admin sends payment instruction/link.
10. Admin updates order once payment is confirmed.
11. Admin approves order for sourcing.
12. Admin updates fulfilment status until delivery/completion.

## Admin Alerts Needed

- New unpaid orders
- Paid orders needing review
- Orders approved for sourcing
- Orders out for delivery
- Orders with reported issues
- Group-buys close to minimum target
- Group-buys that failed minimum target
- Business buyers close to credit limit
