# OneFarmTech

OneFarmTech is a WhatsApp-first farm-to-city procurement platform for Nigeria.

It helps restaurants, hotels, caterers, retailers, food vendors, large households, and bulk buyers source fresh farm produce from trusted rural supply networks.

The project is being built as a managed procurement platform, not an open marketplace.

## MVP direction

- Landing page
- FAQ page
- Buyer login
- Buyer dashboard
- Admin dashboard
- WhatsApp order flow
- Manual order management
- Payment tracking
- Group-buy support
- Pickup and delivery options

## Not included in MVP

- Farmer login
- Open marketplace
- User-created split orders
- Wallet balance
- Delivery API
- Full WhatsApp automation
