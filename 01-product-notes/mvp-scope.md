# OneFarmTech MVP Scope

## MVP must include

- Landing page
- FAQ page
- Login page
- Buyer dashboard placeholder
- Admin dashboard
- WhatsApp order CTA
- Product list
- Customer records
- Manual order creation
- Order status tracking
- Payment status tracking
- Delivery method options
- Pickup locations
- Group-buy preview
- Admin-created group-buy structure

## MVP must not include yet

- Farmer login
- Open marketplace
- User-created split orders
- Wallet balance
- Delivery API
- Escrow
- Reviews/ratings
- Multi-vendor checkout
- In-app chat
- Full WhatsApp Business API automation
